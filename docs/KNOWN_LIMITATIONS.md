# Known Limitations

## Current MVP Constraints

* The demonstration content is intentionally limited to a small learning path.
* User authentication is simplified and currently relies on a shared demo account.
* Duels and Shop are prototype modules and are not fully implemented.
* Quiz functionality is planned for future expansion.
* Achievement and leaderboard systems are currently demonstration-oriented.
* Some dashboard statistics use presentation data rather than fully dynamic calculations.

## Future Development

Planned improvements include:

* User authentication and authorization
* Expanded educational content
* Interactive quizzes
* PvP learning duels
* Shop inventory and purchases
* Advanced achievements and ranking systems
* Automated testing coverage
